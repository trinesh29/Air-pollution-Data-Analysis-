# 🌍 Air Pollution Data Analysis Project

## 📌 Project Overview

This project focuses on analyzing real-world air pollution data to extract meaningful insights using data analysis, visualization, statistical testing, and machine learning techniques.

The goal is to understand pollution patterns, identify trends, detect outliers, and analyze relationships between different pollutant values.

---

## 🔗 Dataset

Source: Public Air Pollution Dataset  

The dataset contains:
- Country, State, City  
- Monitoring Station  
- Pollutant Types (CO, NO2, OZONE, NH3, etc.)  
- Minimum, Maximum, and Average Pollution Values  
- Timestamp (Date & Time)  

---

## 🎯 Objectives

- Perform data cleaning and preprocessing  
- Handle missing values  
- Analyze pollution distribution  
- Study relationships between pollutant values  
- Detect outliers  
- Perform statistical testing  
- Build a machine learning model for prediction  

---

## 📊 Key Analysis Performed

### 1. Data Cleaning
- Removed missing values  
- Converted columns to correct data types  
- Prepared dataset for analysis  

### 2. Pollution Distribution
- Used histogram and violin plot  
- Observed spread and variation in pollution values  

### 3. Correlation Analysis
- Used heatmap to analyze relationships  
- Found strong correlation between min, max, and average values  

### 4. Outlier Detection
- Used boxplot to detect extreme values  
- Identified variability in pollution levels  

### 5. Comparative Analysis
- City-wise pollution comparison using bar chart  
- Distribution using pie chart  

### 6. Trend Analysis
- Created line plot using ordered data  
- Visualized variation across observations  

---

## 🧪 Statistical Analysis

- Z-Test → Measures deviation from expected mean  
- T-Test → Checks statistical significance  
- Shapiro-Wilk Test → Tests data normality  

---

## 🤖 Machine Learning

- Model: Linear Regression  
- Features: pollutant_min, pollutant_max  
- Target: pollutant_avg  

### 📌 Model Performance
- R² Score: ~0.90+  
- Indicates strong prediction accuracy  

---

## 📈 Insights

- Strong relationship between minimum and maximum pollutant levels  
- Pollution values show high variability  
- Data is not normally distributed  
- Machine learning model performs well

This project demonstrates how data analysis, visualization, statistical techniques, and machine learning can be used to extract valuable insights from real-world pollution data.
